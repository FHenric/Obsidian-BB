
## casos de negatividade da potenciação:

Situações que podem aparecer negatividade na potenciação:

1) <span style="color:rgb(255, 0, 0)">Apenas a base está entre parênteses,</span> nesse caso o 2 está potenciando toda a base e isso inclui o bendito do sinal negativo: ATENÇÃO -> o expoente diz o sinal do resultado (<span style="color:rgb(0, 176, 80)">par é positivo</span> e <span style="color:rgb(255, 0, 0)">impar é negativo</span>)
$$
(-5)^2 = -5 * -5 = 25
$$

2) A base E o expoente estão sendo postos em parênteses invisiveis. Nesse caso o sinal não está dentro da potenciação então não deve ser replicado. O loop só ocorre com quem está parametrizado, como no caso acima.
$$
-(5^2) = -5 * 5 = -25
$$

3) -(-2)^2 é quando a base é negativa e por fora ainda tem uma negatividade, pra resolver só ignora o sinal de fora, faz a conta como no caso 1 e aplica o sinal:
$$
-2 * -2 = 4 -> (mais/com/menos = menos) -> -4
$$

4) ($3/2)^2$ é o mesmo que $3^2/2^2$:
3 * 3 = 9 e 2 * 2 = 4 -> 9/4
 
   
5) -(2/3)^3 = -(- 8/27) -> resolve a potenciação e depois se resolve com os - que nesse caso é só juntar o de fora com o de dentro

### Expoente negativo: INVERTE A BASE

Simples meu mano, tu viu que o expoente é negativo, é só inverter a base e o expoente vai junto com a base para o lugar do denominador. Vale observar que o expoente negativo implica um valor menor do que a base em seu resultado como no caso abaixo:

$2^(-2) = 1/2^2$  = 1/4
